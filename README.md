# seamCarving
C++ implementation of size carving/dynamic programming algorithm according to "Seam Carving for Content-Aware Image Resizing" by Shai Avidan and Ariel Shamir.

###Usage
```
.seamCarving [IMAGE_FILE] [REALTIME, 'y' for 'yes']
```
